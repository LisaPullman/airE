# 任务：优化 airE 航空英语学习项目

## 项目背景
airE 是一款面向小学生的航空英语学习应用，使用 React 18 + TypeScript + Vite + Tailwind CSS + Zustand 构建。

## 优化目标（按优先级排序）

### 1. 安全性增强
- 输入验证加强
- XSS 防护
- 敏感信息保护
- API 安全加固

### 2. 易用性提升
- UI/UX 优化
- 响应式设计改进
- 交互流程简化
- 加载性能优化
- 无障碍访问改善

### 3. 学习内容扩展
- 新增航空英语单词（至少50个）
- 新增航空英语句型（至少30个）
- 增加更多情景对话
- 丰富测验题目

## 技术要求
- 保持现有技术栈不变
- 不破坏现有功能
- 保持代码风格一致
- 确保 PWA 离线功能正常

## 完成标准
1. 所有安全漏洞修复
2. 用户体验评分提升
3. 单词库 > 200个
4. 句型库 > 100个
5. 无阻断性bug
