## 1. 项目概述

### 1.1 项目背景

开发一个面向小学生的航空英语学习平台，基于**目标达成系统**框架，通过游戏化的方式激发学习兴趣，帮助小学生在"飞行任务"中掌握航空英语核心句型和词汇。

### 1.2 目标用户

| 用户群体 | 描述 | 主要需求 |
|---------|------|---------|
| 小学生（6-12岁） | 核心用户 | 趣味学习、游戏化激励、成就感、目标驱动 |
| 家长 | 管理者 | 进度跟踪、学习报告、内容管理 |

### 1.3 核心价值

- **不是学英语，是成为飞行员** - 将学习包装为飞行任务训练
- **目标驱动学习** - 设定目标 → 学习 → 测试验证 → 达成成就
- **游戏化激励** - 航空称谓进阶、徽章解锁、成就展示
- **专业内容** - 真实航空英语场景（机场对话、塔台通信等）

### 1.4 产品定位

```
目标达成系统 + 航空英语学习
通过"能力测试目标"验证学习成果，完成目标解锁成就
```

---

## 2. 核心闭环

```
目标设定 → 学习内容 → 互动练习 → 能力测试 → 达成确认 → 解锁成就 → 进阶目标
```

**关键设计原则**:
- 用户设定学习目标（如：掌握Module 1核心词汇）
- 通过能力测试验证是否达成目标
- 达成目标后解锁相应成就和称号

## 3. 功能需求

### 3.1 目标设定模块

**功能描述**: 用户可创建自定义学习目标，设定测试达标分数作为达成条件

**功能点**:
| 功能 | 说明 | 优先级 |
|-----|------|-------|
| 创建目标 | 设定学习目标名称、关联内容、达标分数 | P0 |
| 目标分类 | 句型掌握 / 词汇记忆 / 听力理解 / 对话练习 | P0 |
| 目标状态 | 进行中 / 已达成 / 已放弃 | P0 |
| 目标列表 | 显示所有目标及完成进度 | P0 |
| 目标详情 | 查看目标关联的学习内容和测试要求 | P0 |

### 3.2 学习内容模块

**功能描述**: 内置航空英语基础内容模板，用户可编辑和扩展

**功能点**:
| 功能 | 说明 | 优先级 |
|-----|------|-------|
| 词汇卡片 | 单词+配图+发音按钮 | P0 |
| 句型展示 | 英文句子+中文翻译+音频 | P0 |
| 内容模板 | 内置航空英语基础内容模板 | P0 |
| 内容编辑 | 用户可编辑和扩展学习内容 | P1 |
| 内容分类 | 按模块（飞机认知、塔台对话、天气英语等）组织 | P0 |

### 3.3 互动练习模块

**支持的题型**:
1. 选择题/填空题 - 词汇含义、句型填空
2. 情景对话模拟 - 角色扮演、对话补全
3. 听说练习 - 跟读练习（预留语音识别接口）
4. 趣味游戏 - 卡片匹配、拼词挑战

**功能点**:
| 功能 | 说明 | 优先级 |
|-----|------|-------|
| 题目展示 | 根据题型渲染不同UI | P0 |
| 答案提交 | 记录用户答案 | P0 |
| 即时反馈 | 正确/错误提示 | P0 |
| 进度追踪 | 记录练习完成情况 | P1 |

### 3.4 能力测试模块

**功能描述**: 综合测试，检验目标相关内容的掌握程度

**功能点**:
| 功能 | 说明 | 优先级 |
|-----|------|-------|
| 综合测试 | 随机抽题，覆盖目标相关所有内容 | P0 |
| 测试评分 | 优秀/良好/合格/待提高 | P0 |
| 达标判定 | 达到设定分数即目标达成 | P0 |
| 成绩记录 | 保存测试成绩历史 | P1 |

### 3.5 成就系统

**功能描述**: 航空称谓进阶体系 + 成就徽章

**功能点**:
| 功能 | 说明 | 优先级 |
|-----|------|-------|
| 称号进阶 | 新手飞行员 → 传奇飞行员 | P0 |
| 徽章展示 | 已获得的所有徽章 | P1 |
| 成就动画 | 达成时播放庆祝动画 | P1 |
| 进度显示 | 距离下一称号的进度 | P0 |

### 3.6 进度追踪

**功能点**:
| 功能 | 说明 | 优先级 |
|-----|------|-------|
| 进度条 | 可视化展示目标完成进度 | P0 |
| 学习统计 | 学习时长、词汇/句型掌握度 | P1 |
| 雷达图 | 多维度能力展示 | P1 |
| 目标完成率 | 仪表盘展示目标达成情况 | P0 |

### 3.7 个人中心

**功能点**:
| 功能 | 说明 | 优先级 |
|-----|------|-------|
| 账户设置 | 基本信息管理 | P0 |
| 报告导出 | PDF格式学习报告导出 | P1 |
| 成就展示 | 个人成就墙 | P0 |

## 4. 互动练习模块

### 4.1 选择题/填空题

**描述**: 词汇含义、句型填空、翻译练习

**数据结构**:
```json
{
  "question_type": "choice",
  "question": "What does 'runway' mean?",
  "options": ["跑道", "机翼", "驾驶舱", "螺旋桨"],
  "correct_answer": "跑道"
}
```

### 4.2 情景对话模拟

**描述**: 根据场景选择正确的回应（角色扮演）

**数据结构**:
```json
{
  "question_type": "scenario",
  "scenario": "你是塔台，飞行员说：'Ready for takeoff'",
  "question": "你应该怎么回复？",
  "options": [
    "Cleared for takeoff",
    "Hold position",
    "Request landing",
    "Taxi to runway"
  ],
  "correct_answer": "Cleared for takeoff"
}
```

### 4.3 听说练习

**描述**: 跟读练习，预留语音识别评分接口

**功能点**:
- 播放标准发音音频
- 录音按钮（预留AI评分接口）
- 对比播放原音和跟读录音

### 4.4 趣味游戏

**描述**: 卡片匹配、拼词挑战等趣味练习

**卡片匹配示例**:
```json
{
  "question_type": "matching",
  "question": "请将单词与正确的翻译配对",
  "pairs": [
    {"left": "wing", "right": "机翼"},
    {"left": "cockpit", "right": "驾驶舱"},
    {"left": "runway", "right": "跑道"}
  ]
}
```

## 5. 成就系统

### 5.1 航空称谓进阶体系

| 等级 | 称号 | 解锁条件 | 升级条件 |
|-----|------|---------|---------|
| Lv1 | 新手飞行员 | 注册完成 | 完成1个目标 |
| Lv2 | 初级副驾 | 完成5个目标 | 完成10个目标 |

### 5.2 成就徽章

| 徽章名称 | 获取条件 | 图标 |
|---------|---------|------|
| 塔台小能手 | 完成塔台对话模块 | 📻 |
| 天气预报员 | 完成天气英语模块 | 🌤️ |
| 紧急救援队 | 完成紧急情况模块 | 🚨 |
| 全勤飞行员 | 连续学习7天 | 📅 |
| 满分达人 | 单次测试获得满分 | ⭐ |
| 词汇达人 | 掌握100个航空词汇 | 📚 |

### 5.3 激励机制

| 激励类型 | 说明 |
|---------|------|
| 称号进阶 | 6级航空称谓体系 |
| 徽章解锁 | 完成特定挑战获得 |
| 进度可视化 | 距离下一称号的进度条 |
| 成就动画 | 达成时播放庆祝效果 |

## 6. 课程内容设计

### 6.1 三大课程模块

#### Module 1: 飞机认知 (入门)

**核心词汇**:
| 英文 | 中文 | 配图 |
|-----|------|------|
| wing | 机翼 | 飞机侧视图标注 |
| cockpit | 驾驶舱 | 驾驶舱特写 |
| runway | 跑道 | 机场俯视图 |
| engine | 发动机 | 发动机位置图 |
| fuselage | 机身 | 飞机整体图 |
| tail | 尾翼 | 飞机尾部 |
| landing gear | 起落架 | 起落架结构图 |

**核心句型**:
- Where is the gate?
- How do I get to the terminal?
- I need to check in first.

#### Module 2: 机场流程 (进阶)

**核心词汇**:
| 英文 | 中文 | 场景 |
|-----|------|------|
| terminal | 航站楼 | 值机 |
| boarding pass | 登机牌 | 值机 |
| security | 安检 | 安检 |
| gate | 登机口 | 登机 |
| luggage | 行李 | 托运 |

**核心句型**:
- Can I have a window seat?
- Where is the security check?
- What time does the flight board?

#### Module 3: 塔台通信 (高级)

**核心句型**:
| 英文 | 中文 | 场景 |
|-----|------|------|
| Ready for takeoff | 准备起飞 | 起飞请求 |
| Request landing | 请求降落 | 降落请求 |
| Cleared to land | 准许降落 | 塔台回复 |
| Taxi to runway | 滑行至跑道 | 地面指令 |
| Contact tower | 联系塔台 | 交接指令 |

### 6.2 内容结构规范

```json
{
  "module_id": "M1",
  "module_name": "飞机认知",
  "order": 1,
  "vocabularies": [
    {
      "word": "wing",
      "translation": "机翼",
      "image_url": "/assets/images/wing.jpg",
      "audio_url": "/assets/audio/wing.mp3",
      "example_sentence": "The wing helps the plane fly."
    }
  ],
  "sentences": [
    {
      "english": "Where is the gate?",
      "chinese": "登机口在哪里？",
      "audio_url": "/assets/audio/gate.mp3"
    }
  ],
  "target_config": {
    "min_score": 80,
    "question_count": 10,
    "question_types": ["choice", "matching", "scenario"]
  }
}
```

## 11. 验收标准

### 11.1 功能验收

| 模块 | 验收标准 |
|-----|---------|
| 目标设定 | 用户可创建、编辑、删除学习目标 |

- [ ] PWA离线功能正常
