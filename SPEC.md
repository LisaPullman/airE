# 航空英语学习目标达成系统 - 规格说明

## 1. 项目概述

**项目名称**: AirE - 航空英语学习目标达成系统
**目标用户**: 小学生（6-12岁）
**核心价值**: 通过目标驱动的学习系统，帮助小学生掌握航空英语核心句型和词汇
**产品定位**: 基于浏览器的完整学习系统，包含目标设定、进度追踪、能力测试、成就解锁

## 2. 核心闭环

```
目标设定 → 学习内容 → 进度追踪 → 能力测试 → 达成确认 → 解锁成就 → 进阶目标
```

## 3. 功能规格

### 3.1 目标设定模块
- 用户可创建自定义学习目标
- 支持设定测试达标分数作为达成条件
- 目标分类：句型掌握 / 词汇记忆 / 听力理解 / 对话练习
- 目标状态：进行中 / 已达成 / 已放弃

### 3.2 学习内容模块
- 内置航空英语基础内容模板
- 用户可编辑和扩展学习内容
- 内容类型：
  - 核心句型（问路、登机、点餐、紧急情况等）
  - 航空词汇（飞机部件、机场设施、天气术语等）
  - 情景对话（机场Check-in、机上服务、塔台通信等）

### 3.3 互动练习模块
- 选择题：词汇含义、句型填空
- 填空题：补全句子、翻译练习
- 情景对话模拟：角色扮演、对话补全
- 听说练习：跟读练习、语音识别评分
- 趣味游戏：卡片匹配、拼词挑战

### 3.4 能力测试模块
- 综合测试：涵盖目标相关所有内容
- 随机抽题，检验真实掌握程度
- 测试评分：优秀/良好/合格/待提高
- 达标判定：达到设定分数即目标达成

### 3.5 成就系统
- 航空称谓进阶体系：
  - 新手飞行员 → 初级副驾 → 副驾 → 机长 → 指挥官 → 传奇飞行员
- 成就徽章：完成特定挑战获得
- 成就展示：个人档案页展示全部成就
- 解锁动画：达成时播放庆祝动画

### 3.6 进度追踪
- 可视化进度条
- 学习时长统计
- 词汇/句型掌握度雷达图
- 目标完成率仪表盘

## 4. 技术规格

### 4.1 技术栈
- **前端框架**: React 或 Vue（组件化开发）
- **构建工具**: Vite / Create React App / Vue CLI
- **状态管理**: React Context / Vuex / Pinia
- **样式方案**: Tailwind CSS / CSS Modules
- **图标库**: Lucide / Heroicons

### 4.2 离线支持
- **PWA支持**: Service Worker 实现离线缓存
- **离线功能**: 已学习内容可离线复习
- **数据同步**: 在线时自动同步到服务器（未来扩展）

### 4.3 数据存储
- **用户系统**: 简单注册（邮箱/用户名）
- **本地存储**: LocalStorage 缓存用户偏好
- **云端存储**: 用户数据存储在服务器数据库
- **数据导出**: 支持导出 PDF 格式学习报告

### 4.4 响应式设计
- **桌面优先**: 主要适配笔记本/台式机
- **平板适配**: 响应式布局适配 iPad
- **移动端**: 基础功能可用，优先保证桌面体验

## 5. UI/UX 设计规格

### 5.1 视觉风格
- **设计主题**: 航空主题
- **设计元素**: 飞机、机场、驾驶舱元素
- **视觉特点**: 扁平化设计 + 适度的拟物元素

### 5.2 配色方案
- **主题色**: 航空蓝（#1E40AF）、天空蓝（#60A5FA）
- **辅助色**: 橙色（#F97316，用于强调/按钮）、绿色（#22C55E，用于成功/达成）
- **配色模式**: 系统自适应（明暗模式自动切换）
- **无障碍**: 符合 WCAG 2.1 AA 标准

### 5.3 字体规范
- **正文字体**: 思源黑体 / Inter（儿童友好，易读）
- **标题字体**: 思源黑体 Bold
- **字号**: 正文16px+，标题24px+

### 5.4 界面布局
- **首页仪表盘**: 目标概览、今日任务、进度统计
- **目标页面**: 目标列表、创建目标、目标详情
- **学习页面**: 内容展示、练习区域、测试入口
- **成就页面**: 成就展示、徽章墙、称号进度
- **个人中心**: 设置、报告导出、账户管理

## 6. 部署方案

### 6.1 免费部署平台（多平台部署）
- **GitHub Pages**: 主部署平台，免费静态托管
- **Vercel**: 边缘网络，快速加载
- **Cloudflare Pages**: 全球CDN，无限流量
- **Netlify**: 简单配置，自动HTTPS

### 6.2 部署策略
- **CI/CD**: GitHub Actions 自动构建部署
- **自定义域名**: 支持配置自定义域名
- **SSL**: 自动HTTPS证书

## 7. 内容规划

### 7.1 内置内容模板
- **第一阶段（入门）**:
  - 机场常用词汇：terminal, gate, runway, boarding...
  - 基础问路句型：Where is...? / How do I get to...?
- **第二阶段（进阶）**:
  - 登机流程：Check-in, security, customs...
  - 机上对话：Menu,枕头、毛毯、饮料...
  - 紧急情况：Emergency, evacuation...
- **第三阶段（高级）**:
  - 塔台通信：Tower, runway, clearance...
  - 天气术语：Turbulence, visibility, wind...

### 7.2 内容来源
- 系统内置基础内容模板
- 用户可编辑、扩展学习内容
- 支持导入JSON格式的内容包

## 8. 风险与应对

### 8.1 主要风险
- **趣味性问题**: 孩子觉得无聊不想用
  - 应对: 加入成就系统、趣味游戏、情景对话模拟
- **使用率问题**: 做完后成为摆设
  - 应对: 设计有吸引力的视觉风格，每日任务提醒

### 8.2 权衡取舍
- **功能边界**: MVP + 扩展空间，初期实现完整学习系统
- **开发投入**: 中期打磨（2-4周）
- **内容策略**: 共建内容，系统提供模板，用户可扩展
